Indexed DB

This example is using plain vanilla JS and core indexedDB methods.

If you are familiar with document databases like MongoDB, you can think of an ObjectStore as a collection.

The Core Object and Interfaces are:

  • window.indexedDB => The database object holds Object Stores.
  • IDB ObjectStore => An Object Store inside a database. ( A Collection of objects )
  • IDB Request => A request object for add, put, delete, etc.
  • IDB Transaction => A wrapper around requests.
  • IDB Index => An index added to an Object Store
  • IDB Cursor => To track current position inside the results of a request.
  • IDB KeyRange => A range of values for matching a key.
